Connect Microsoft to Circe

This takes about 5 minutes and only needs to be done once. Follow each step below.

1

Go to the Microsoft Azure Portal

Click the button below. Sign in with the same Microsoft or Outlook account you want Circe to use. It is free — no payment needed.

Open Azure Portal
2

Find App Registrations

In the search bar at the very top of the page, type App registrations and click it when it appears in the results.

3

Create a new app

Click New registration at the top left.

For the name, type Circe.

Under "Supported account types", choose Personal Microsoft accounts only.

Leave everything else as-is and click Register.

4

Add a redirect address

After registering, you'll see the app's Overview page. In the left sidebar, click Authentication.

Click Add a platform, then choose Single-page application.

In the Redirect URI box, type: http://localhost:3000

Click Configure, then click Save at the top.

5

Copy your Client ID

Click Overview in the left sidebar.

You'll see Application (client) ID — it's a long string of letters and numbers with dashes.

Copy that entire ID — it looks something like: a1b2c3d4-1234-5678-abcd-ef1234567890

Paste your Client ID here

Saved! Taking you back to Circe...
