✦ Connect Google to Circe

This takes about 5 minutes and only needs to be done once. Follow each step below.

1

Go to Google Cloud Console

Click the button below. Sign in with the same Gmail account you want Circe to use. It's free — no payment needed.

Open Google Cloud Console ↗
2

Create a new project

At the top of the page, click the project dropdown (it might say "My First Project" or "Select a project").

Click New Project. Name it Circe. Click Create. Wait a few seconds for it to finish.

Make sure "Circe" is selected in the project dropdown before moving on.

3

Enable the Calendar, Tasks, and Gmail APIs

In the left sidebar, click APIs & Services → Enable APIs and Services.

Search for and enable each of these (click each one, then click Enable):

• Google Calendar API
• Tasks API
• Gmail API

4

Set up the consent screen

In the left sidebar, click APIs & Services → OAuth consent screen.

Choose External and click Create.

Fill in App name: Circe and your Gmail address for the support email. Scroll down and click Save and Continue three times until you reach the Summary screen. Click Back to Dashboard.

5

Create your credentials

In the left sidebar, click APIs & Services → Credentials.

Click + Create Credentials → OAuth 2.0 Client ID.

For Application type, choose Web application. Name it Circe.

Under Authorized JavaScript origins, click + Add URI and type: http://localhost:3000

Click Create.

A box will pop up showing your Client ID — it's a long string ending in .apps.googleusercontent.com. Copy that whole thing.

✓ Paste your Client ID here

✓ Saved! Taking you back to Circe…
