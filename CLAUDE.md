# Circe — Kate's Personal Assistant App

## What this is
A voice-activated web app for Kate, a special education teacher (grades 7-12)
with early onset Alzheimer's. The assistant is named Circe, activated by saying
"Hey Circe". It always responds out loud.

## Features
- Wake word detection ("Hey Circe")
- Task manager, student tracker, schedule, writing helper
- Circe consults a Claude Opus advisor when unsure
- Saves data to localStorage

## Current model setup
- Circe uses claude-sonnet-4-6 for fast replies
- The consultant uses claude-opus-4-6 for hard questions
